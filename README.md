# To-Do List Application

This is a C++ project that implements a simple command-line to-do list application.

## Features
- Add tasks
- Remove tasks by ID
- View the task list

## How to Run
1. Compile the code:
   ->bash
   g++ -I include src/main.cpp src/task.cpp src/todo.cpp -o todo_list
2. Run the exe:
   ->
   ./todo_list.exe
